# Better.com Clone
## Overview
A multi-page, responsive web app replicating Better.com built with React & Next.js.

## Features
- Home, About Us, Mortgage Calculator, Start Page
- Responsive design
- Mortgage Calculator logic
- Styled using CSS Modules
- Deployed on Vercel
